# Podium
Podcast Network
===============

![Logo](client/src/assets/podium-text.png)

A place to build RSS feeds and host Podcast content.

A built in DAW for simple sound manipulation.
    -   Multi track 
    -   Add, cut, paste loops 
    -   Bounce audio
    -   Push bounces direct to storage
